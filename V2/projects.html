Projects

Here are some of the random stuff I've worked on

EV Charging Station

For my home I needed a reliable EV charging station. I researched various options and ultimately decided on a older charging station from EVBox. These could easily be modified to by replacing the internal controller with something like a SmartEVSE. After this I also added RFID authentication to the setup so that anyone with an public charging card can charge at my home. And finally I ofcourse made sure it was NEN Compliant by adding additional safety features.

Small init changes

For one of the packages that I used on Alpine Linux there was a small issue with the init script, Due to some internal changes in the package some of the flags were no longer valid. I updated the init script to reflect these changes and ensure that the package would start correctly.
